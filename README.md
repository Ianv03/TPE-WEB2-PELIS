## Tematica
    Peliculas con su respectiva informacion, clasificada por generos.

## Descripcion
    Este sitio permite mostrar diferentes peliculas con su titulo, portada, lanzamiento, genero y descripcion de la misma.

## Modelo de Datos
    Esta compuesto por:
        Peliculas: Dentro se incluye tanto el titulo como la portada, el lanzamiento y una descripcion
        Generos: Para clasificar las diferentes peliculas
